Class NoTableMapping

java.lang.Object
  net.sf.persist.Mapping
      net.sf.persist.NoTableMapping

public class NoTableMapping
extends Mapping

Represents the mapping of columns to getters and setters of a POJO.

It is used when a class specifies a NoTable annotation, which means the class is not mapped to a table in the database, and will be only used to store data from queries.
